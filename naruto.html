L'html è un linguaggio markup con cui viene definita la struttura base di una pagina web.

✬ ✬ CSS ✬ ✬

Il CSS è un altro linguaggio web che serve per personalizzare l'aspetto degli elementi HTML.
Ecco le principali caratteristiche:

  • L'ultima regola di solito vince sulle altre
  • Alcuni selettori vincono anche se sono in cima
  • Esistono variabili per fare modifiche più rapidamente
  • Le classi sono importanti!!!
